# 🌐 旅冬亦的个人网站
欢迎访问我的个人网站！这是一个展示个人信息、经历和联系方式的静态网页项目。网站采用响应式设计，支持深色/浅色主题切换，并包含更新日志。

## 🌟 核心功能

### 主页
- **响应式设计**：适配桌面和移动设备
- **动态头像**：悬停时放大并显示蓝色边框
- **个人信息展示**：包含姓名、标题、简介和联系方式
- **自定义字体**：使用 `MiSans` 增强视觉效果

### 更新日志
- **主题切换**：支持深色/浅色模式
- **版本历史**：清晰展示每次更新的日期、版本号和变更内容
- **语义化标签**：使用表情符号标识变更类型

## 🎨 设计特色

1. **视觉风格**
    - 圆形头像设计
    - 卡片式布局
    - 平滑的悬停动画效果

2. **响应式设计**
    - 移动端优先的媒体查询
    - 自适应容器宽度
    - 触摸友好的交互元素

3. **可访问性**
    - 语义化 HTML 标签
    - 高对比度文本
    - 键盘导航支持

## 🛠 技术栈
- **HTML5**：语义化标签结构
- **CSS3**：
    - Flexbox 布局
    - CSS 变量
    - 过渡动画
    - 自定义字体
- **JavaScript**：年份自动更新

## 📱 手机端支持
本网页支持**Apk**文件，提供两种类型：
<br>
- **在线版 Online**：即网页映射至软件，网页和软件同步更新信息。
  - 在线版(已弃项)：~~https://github.com/lxj13533251583/ldy/releases/tag/Online/~~
- **离线版 Offline**：打包网页源文件至软件，可离线查看，但更新信息需要在本链接下载。
  - 离线版(构建中)：https://github.com/lxj13533251583/ldy/releases/tag/Offline/
